# 72-Hour Test Loop SOP

## Day 0: Publish
- Publish 20 listings/day from listing-pack-v1.csv
- Ensure mockups are clean, readable, high contrast
- Price baseline: Tee 24.99, Mug 16.99, Sticker 4.99

## Day 1: QA
- Check listing status and image quality
- Fix typos, remove clutter, ensure keyword clarity

## Day 3 (72h): Decision
For each listing:
1) KILL if no traction and poor relevance
2) ITERATE if some clicks but no orders (change title/main image/price)
3) SCALE if orders + positive margin

## Scale action
- Clone winning message into 5 variants:
  - different colorways
  - different product types
  - recipient variants (mom/dad/grandma)
