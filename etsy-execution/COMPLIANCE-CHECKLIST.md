# Etsy POD Compliance Checklist (No IP Risk)

## Before publishing any listing
- [ ] No brand names, logos, team names, movie/game characters, or celebrity references.
- [ ] No copyrighted quotes/lyrics.
- [ ] Original wording and original visual composition.
- [ ] Production partner disclosed in Etsy (Printify/Printful/etc).
- [ ] Product description matches actual item and print method.
- [ ] No prohibited categories (weapons, hate, explicit sexual content, illegal items).

## Listing QA
- [ ] Title readable and not keyword stuffed
- [ ] 13 tags used, relevant to buyer intent
- [ ] Main mockup is legible at thumbnail size
- [ ] Price supports >= 30% estimated margin
- [ ] Shipping profile correctly mapped

## Escalation
If uncertain about IP/trademark, DO NOT publish; move to review queue.
